# Produisez-une-tude-de-march-
Produisez une étude de marché 
## Contexte du projet
*L’objectif principal de ce travail est d’identifier des profils de pays à partir de variables démographiques, économiques, alimentaires et commerciales, afin de mieux comprendre quels marchés semblent les plus attractifs.
## Problématique
* Comment regrouper les pays en marchés homogènes ?
* Quels facteurs influencent l’attractivité d’un marché ?
* Quels marchés doivent être ciblés en priorité ?
